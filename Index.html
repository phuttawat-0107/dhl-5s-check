📖 คู่มือหลักการ 5S
1S — สะสาง · SORT · SEIRI
แยกของที่จำเป็นออกจากไม่จำเป็น
📌 ตรวจสอบทุกรายการในพื้นที่ทำงาน
📌 แยกประเภท: ใช้งาน / ไม่ได้ใช้ / ยังไม่แน่ใจ
📌 กำจัดของที่ไม่จำเป็นออกจากพื้นที่ทันที
📌 ติดป้าย Red Tag กับของที่รอตัดสินใจ
📌 ไม่สะสมพัสดุ / อุปกรณ์โดยไม่จำเป็น
✅ ผ่าน: ไม่มีของที่ไม่ใช้วางในพื้นที่ปฏิบัติงาน
💡 ถามตัวเองว่า "ใช้ในเดือนนี้ไหม?" ถ้าไม่ → เอาออก
2S — สะดวก · SET IN ORDER · SEITON
จัดวางของที่จำเป็นให้เป็นระเบียบ
📌 ทุกชิ้นต้องมี "บ้าน" ของตัวเอง
📌 ติดป้ายและกำหนดตำแหน่งวางที่ชัดเจน
📌 ใช้หลัก FIFO กับพัสดุ (First In First Out)
📌 จัดให้หยิบง่าย ใช้งานได้รวดเร็ว
📌 วาดเส้นกำหนดพื้นที่ (Line Marking) ให้ชัดเจน
✅ ผ่าน: ทุกอย่างมีป้ายบอก หยิบได้ใน 30 วินาที
💡 ถ้าต้องถามว่าวางไว้ไหน = ยังไม่สะดวก
3S — สะอาด · SHINE · SEISO
ทำความสะอาดและตรวจสอบอุปกรณ์
📌 ทำความสะอาดพื้น โต๊ะ อุปกรณ์ทุกวัน
📌 ตรวจสอบอุปกรณ์ขณะทำความสะอาด
📌 แจ้งซ่อมทันทีเมื่อพบชำรุดหรือผิดปกติ
📌 ทำความสะอาดหลังใช้งานทุกครั้ง
📌 กำหนดตารางเวรทำความสะอาดประจำวัน
✅ ผ่าน: พื้นที่สะอาด ไม่มีคราบ ไม่มีขยะ
💡 สะอาด = ตรวจสอบด้วย ไม่ใช่แค่กวาด
4S — สุขลักษณะ · STANDARDIZE · SEIKETSU
รักษามาตรฐาน 3S แรกให้สม่ำเสมอ
📌 สร้าง SOP และ Visual Management ในพื้นที่
📌 ทำให้ 3S แรกเป็นเรื่องปกติ ไม่ใช่พิเศษ
📌 ติดภาพ Before–After เป็นมาตรฐานอ้างอิง
📌 ทบทวนและอัพเดต SOP อย่างสม่ำเสมอ
📌 ทุกคนในทีมรู้มาตรฐานเดียวกัน
✅ ผ่าน: มี SOP / ป้ายมาตรฐานชัดเจนในพื้นที่
💡 ถ้าต้องบอกวาจา = ยังไม่ Standardize
5S — สร้างนิสัย · SUSTAIN · SHITSUKE
รักษาวินัย ทำ 5S ให้เป็นส่วนหนึ่งของงาน
📌 ทำ 5S ทุกวันโดยไม่ต้องรอให้สั่ง
📌 รายงานปัญหาที่พบทันที ไม่ปล่อยผ่าน
📌 ให้ความร่วมมือกับทีมอย่างสม่ำเสมอ
📌 เป็นแบบอย่างที่ดีให้เพื่อนร่วมงาน
📌 เข้าร่วมการตรวจ 5S ประจำเดือนทุกครั้ง
✅ ผ่าน: ทำ 5S ได้โดยไม่ต้องรอให้บอก
💡 5S ที่ดีที่สุด = ทำเองโดยสมัครใจ
